# WhatsMyScore2 - NativeScript
What's My Score App built with NativeScript + Angular 2 + TypeScript 

## Getting Started
Make sure you have all the tools to run [NativeScript](http://docs.nativescript.org/start/getting-started) 

1. npm install 
2. tns run android
